# game_panji
Aplikasi catur berbasis web sederhana.

## Cara menjalankan
1. Buka `index.html` di browser.
2. Klik bidak untuk memilih.
3. Klik kotak tujuan untuk memindahkan.

## Fitur
- Papan catur interaktif 8x8
- Logika gerakan bidak dasar
- Sejarah langkah dan bidak tertangkap
- Reset permainan
